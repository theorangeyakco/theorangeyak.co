Lick.gg

Lick.gg is a social forum and community website. The Orange Yak worked with an in-house development team who were building the backend and integrated that backend with a complete frontend app built on the nuxt.js framework using vue.js. OYC is always excited to work with other developers!

Snazz

Snazz is a market place that provides a one stop solution for all buyers and connects growing business to sellers. We built Snazz's complete e-commerce platform using Shopify on the backend, developed a cohesive design, interactive frontend, and branding material for this incipient start-up.

Youth Policy Collective

Youth Policy Collective is a non-profit think tank. YPC used the Orange Yak's social intiative plan for this cost-effective website. The site includes an attractive promotional section and a fully fledged blog controlled by an Admin panel. Built using Bootstrap 4, Django, Javascript.

Sorobrain

An online French learning platform built for Sorobrain- a small business based in Gurgaon. Built using Django, jQuery, and Bootstrap the platform has a complete authentication system, online competitions, quizzes, and workshops.

StraightZero

StraightZero is a complete Education Resource Delivery software. It allows teachers to sign up to create and sell online courses. It supports online video, tests, resources, and interactive notes. Straighzero currently provides courses for various state-level civil service examinations. Built in Django using jQuery, Vue.js, and Bootstrap.

Roshambo

mcow.ml
